# Speak-with-AI-about-your-Data-Using-Real-Time-Speech-Transcription
**Project Overview**
You will build an app where:
User speaks a question
Speech is converted to text (Speech-to-Text)
AI searches & understands your data
AI generates an answer
Answer is spoken back (Text-to-Speech)
**Tech Stack **
Component            	        Tool
Programming Language	        Python
Speech to Text	              OpenAI Whisper
AI Model	                    OpenAI GPT
Text to Speech	              pyttsx3
Data Source	                  CSV / Text file
UI (optional)	                Streamlit
